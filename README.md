# Arrive. 💳
### Master Your Financial Future

Arrive is a high-performance, premium iOS financial management application designed with a **Liquid Glass** aesthetic. It provides a seamless experience for tracking balances, managing multiple cards, monitoring investments, and gaining intelligent insights into your spending habits.

## Features

- **Liquid Glass UI**: A state-of-the-art interface featuring glassmorphism, dynamic animations, and a sleek dark mode.
- **Card Management**: View and manage Physical, Virtual, and Business cards in a beautiful 3D stack.
- **Smart Insights**: Interactive donut charts and categorized spending activity to help you understand your finances.
- **Investment Watchlist**: Real-time tracking for stocks, ETFs, and Crypto.
- **Quick Actions**: Instant access to Send, Request, and Add funds.
- **Floating Dock**: A custom-built, interactive navigation experience.

## Tech Stack

- **Framework**: SwiftUI
- **Architecture**: MVVM
- **Animations**: SwiftUI Transitions & Spring Animations
- **Design System**: Custom Liquid Glass Design System
- **Assets**: SF Symbols & Custom Vectors

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/Arrive.git
   ```
2. **Open in Xcode**:
   ```bash
   cd Arrive
   open Arrive.xcodeproj
   ```
3. **Run the app**: Select your preferred simulator and hit `Cmd + R`.

## Design Philosophy

Arrive is built on the principle of **Visual Excellence**. Every interaction is designed to feel "alive"—from the springy transitions of the card stack to the blurred translucency of the floating dock. It's not just a tool; it's a premium experience.
